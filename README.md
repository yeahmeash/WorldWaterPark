# waterworldpark

This is a sample Android application built with Kotlin/Java using Android Studio.

## Features
- User authentication
- Real-time data sync
- Clean architecture (MVVM)

## Tech Stack
- Kotlin / Java
- Jetpack Compose / XML
- Firebase / Room

## Getting Started
1. Clone the repo
2. Open in Android Studio
3. Run the app on emulator or device
